# چاپ مربع (Print Square)
---
## صورت مسئله:

**محدودیت زمان:** ۱ ثانیه
**محدودیت حافظه:** ۲۵۶ مگابایت

برنامه‌ای بنویسید که عدد `n` را از ورودی دریافت کرده و سپس یک مربع توخالی به طول و عرض `n` با کاراکتر ستاره (*) چاپ کند.

**ورودی**

در یک خط عدد `n` به شما داده می‌شود.
3 ≤ n ≤ 10

**خروجی**

مربع حاصل را در خروجی چاپ کنید.

**مثال**

**ورودی نمونه ۱:**
```
6
```

**خروجی نمونه ۱:**
```
******
*    *
*    *
*    *
*    *
******
```

---

**توضیحات:**

در این مسئله، شما باید یک مربع توخالی با استفاده از کاراکتر ستاره (*) چاپ کنید.  مربع باید به طول و عرض `n` باشد، که در آن `n` از ورودی دریافت می‌شود.  توجه داشته باشید که فقط لبه‌های مربع باید با ستاره پر شوند، و قسمت داخلی باید خالی بماند (با فاصله).  برای چاپ این مربع، می‌توانید از حلقه‌های تودرتو استفاده کنید.


---
## توضیح راه حل :
1. ورودی از کاربر دریافت میشه
2. با یک حلقه n تایی خط اول چاپ شده
3. با دو حلقه تو در تو خط های میانی چاپ شده(حلقه ها تا n-2)
4. با یک حلقه تا n خط آخر چاپ میشود
