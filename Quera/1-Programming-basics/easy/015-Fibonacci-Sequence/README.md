# دنباله‌ی فیبوناچی (Fibonacci Sequence)

* **محدودیت زمان:** ۲ ثانیه
* **محدودیت حافظه:** ۲۵۶ مگابایت

برنامه‌ای بنویسید که به ازای دریافت جملات $N$ام و $N+1$ام از دنباله فیبوناچی (با فرض $A_0=A_1=1$) و ارسال این دو به تابعی با امضای:

```cpp
void ShowFibNth(long int n, long int n+1)
```

جملات سری فیبوناچی را از جمله $N$ام تا صفرم به صورت معکوس چاپ کند.

توجه کنید که تابع مذکور در هر بار فراخوانی خود، وظیفه چاپ پارامتر اول و فراخوانی مجدد خود (به صورت بازگشتی) را خواهد داشت. بدیهی است که استفاده از حلقه غیرمجاز است.

## ورودی
در خط اول جمله $n$ام و در خط بعد عدد $n+1$ام به شما داده می‌شود. اعداد از ۱,۰۰۰,۰۰۰ کوچکترند.

## خروجی
جملات فیبوناچی را به ترتیب چاپ کنید.

## مثال‌ها

### ورودی نمونه ۱
```text
5
8
```
### خروجی نمونه ۱
```text
5
3
2
1
1
```

### ورودی نمونه ۲
```text
1
1
```
### خروجی نمونه ۲
```text
1
```
