#  کامل بودن یا نبودن (Perfect Number)
---
## صورت مسئله:

**محدودیت زمان:** ۲ ثانیه
**محدودیت حافظه:** ۲۵۶ مگابایت

برنامه‌ای بنویسید که عددی مانند `n` را از کاربر دریافت کند و در صورتی که خاصیت «کامل» بودن را داشته باشد، YES و در غیر این صورت NO را چاپ کند.

یک عدد صحیح و مثبت مثل `n` «کامل» است اگر و تنها اگر مجموع مقسوم‌علیه‌های `n` (غیر از خود `n`) برابر با `n` باشد.

**ورودی**

در یک خط عدد `n` به شما داده می‌شود.
2 ≤ n ≤ 200,000

**خروجی**

چنان چه عدد کامل بود YES در غیر این صورت NO چاپ کنید.

**مثال‌ها**

**ورودی نمونه ۱:**
```
27
```

**خروجی نمونه ۱:**
```
NO
```

مقسوم‌علیه‌های `27` که از آن کوچک‌تر است؛ `1`, `3` و `9` هستند که مجموع آن‌ها

`1 + 3 + 9 = 13 ≠ 27`

است. پس `27` کامل نیست.

**ورودی نمونه ۲:**
```
6
```

**خروجی نمونه ۲:**
```
YES
```

مقسوم‌علیه‌های `6` که از آن کوچک‌تر است؛ `1`, `2` و `3` هستند و مجموع آن‌ها

`1 + 2 + 3 = 6`

است. پس `6` کامل است.

---
## توضیح راه حل :
باید داخل یک حلقه اعدادی که باقی مانده `n` بر آنها صفر است رو پیدا کنیم (مقسوم علیه ها )
داخل همین حلقه بلافاصه اونها رو با هم جمع میکنیم
اگر جمع حاصل با `n` برابر بود پس `n` کامل است
