# فاکتوریل (Factorial)
---
## صورت مسئله:
**محدودیت زمان:** ۱ ثانیه
**محدودیت حافظه:** ۲۵۶ مگابایت

برنامه‌ای بنویسید که عدد `n` را از ورودی گرفته و فاکتوریل آن را محاسبه کرده و نمایش دهد.

منظور از فاکتوریل یک عدد مانند `n` که با نماد `n!` نشان داده می‌شود؛ ضرب اعداد ۱ تا `n` است. به عبارت دیگر:

`n! = 1 × 2 × 3 × ... × n`

**ورودی**

در یک خط عدد `n` به شما داده می‌شود.
1 ≤ n ≤ 10

**خروجی**

مقدار `n` فاکتوریل را در خروجی چاپ کنید.

**مثال**

**ورودی نمونه ۱:**
```
5
```

**خروجی نمونه ۱:**
```
120
```

`5! = 1 × 2 × 3 × 4 × 5 = 120`
