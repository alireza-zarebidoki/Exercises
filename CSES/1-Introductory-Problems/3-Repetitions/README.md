# تکرارها (Repetitions)
---
## صورت مسئله:
**زمان مجاز:** ۱.۰۰ ثانیه
**حافظه مجاز:** ۵۱۲ مگابایت

یک دنباله DNA به شما داده شده است: رشته‌ای متشکل از کاراکترهای A، C، G و T. وظیفه شما یافتن طولانی‌ترین تکرار در این دنباله است. این یک زیررشته با حداکثر طول است که فقط شامل یک نوع کاراکتر باشد.

**ورودی**

تنها خط ورودی شامل رشته‌ای از n کاراکتر است.

**خروجی**

یک عدد صحیح چاپ کنید: طولانی‌ترین تکرار.

**محدودیت‌ها**

1 ≤ n ≤ 10<sup>6</sup>

**مثال**

**ورودی:**
```
ATTCGGGA
```

**خروجی:**
```
3
```
