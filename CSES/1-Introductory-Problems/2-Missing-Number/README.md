# عدد گمشده (Missing Number)

**زمان مجاز:** ۱.۰۰ ثانیه
**حافظه مجاز:** ۵۱۲ مگابایت

به شما تمام اعداد بین ۱، ۲، ...، n به جز یک عدد داده شده است. وظیفه شما پیدا کردن عدد گمشده است.

**ورودی**

خط اول ورودی شامل یک عدد صحیح n است.
خط دوم شامل n-1 عدد است. هر عدد متمایز و بین ۱ تا n (شامل) است.

**خروجی**

عدد گمشده را چاپ کنید.

**محدودیت‌ها**

2 ≤ n ≤ 2 * 10<sup>5</sup>

**مثال**

**ورودی:**
```
5
2 3 1 5
```

**خروجی:**
```
4
```
