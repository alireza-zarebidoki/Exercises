# الگوریتم عجیب (Weird Algorithm)
---
## صورت مسئله:
**زمان مجاز:** ۱.۰۰ ثانیه
**حافظه مجاز:** ۵۱۲ مگابایت

یک الگوریتم را در نظر بگیرید که یک عدد صحیح مثبت n را به عنوان ورودی دریافت می‌کند. اگر n زوج باشد، آن را بر دو تقسیم می‌کند و اگر n فرد باشد، آن را در سه ضرب کرده و یک واحد به آن اضافه می‌کند. این الگوریتم تکرار می‌شود تا زمانی که n برابر با یک شود. برای مثال، دنباله برای n=3 به شرح زیر است:

$$ 3 \rightarrow 10 \rightarrow 5 \rightarrow 16 \rightarrow 8 \rightarrow 4 \rightarrow 2 \rightarrow 1$$

وظیفه شما شبیه‌سازی اجرای این الگوریتم برای یک مقدار داده شده از n است.

**ورودی**

خط ورودی تنها شامل یک عدد صحیح n است.

**خروجی**

یک خط که شامل تمام مقادیر n در طول الگوریتم باشد، چاپ کنید.

**محدودیت‌ها**

1 ≤ n ≤ 10<sup>6</sup>

**مثال**

**ورودی:**
```
3
```

**خروجی:**
```
3 10 5 16 8 4 2 1
```

## توضیح راه حل :

یک عدد به عنوان ورودی از کاربر دریافت میشه .
اگر فرد بود ضربدر سه بعد به اضافه یک
اگر زوج بود تقسیم بر دو
تا در نهایت به عدد یک برسیم
تمامی این فرایند چاپ بشه
